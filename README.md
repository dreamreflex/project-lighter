# Project Lighter - 项目启动器

一个基于 Electron 的多项目一键启动管理工具，支持同时启动和管理多个前后端项目。

## 功能特性

- 一键启动多个项目
- 支持单个项目配置多条命令，按顺序执行
- 独立停止每个项目
- 通过配置文件管理项目
- 实时查看项目输出日志
- 启动时显示 PowerShell 版本信息
- 现代化的用户界面
- 仅支持 Windows 系统

## 安装与运行

1. 安装依赖：
```bash
npm install
```

2. 启动应用：
```bash
npm start
```

## 配置说明

配置文件位于用户数据目录中，首次运行时会自动创建。

### 配置文件格式 (JSON)

支持配置多个项目，每个项目可以包含一条或多条命令。

```json
{
  "projects": [
    {
      "id": "1",
      "name": "示例项目",
      "commands": [
        {
          "name": "步骤1",
          "command": "echo '开始启动'"
        },
        {
          "name": "步骤2",
          "command": "npm run dev"
        }
      ],
      "workingDir": "D:\\projects\\demo"
    }
  ]
}
```

### 字段说明

- `id`: 项目唯一标识符
- `name`: 项目显示名称
- `commands`: 命令列表，按顺序执行
- `workingDir`: 工作目录

## 使用方法

1. **启动项目**：点击"启动"按钮，按顺序执行配置的命令。
2. **停止项目**：点击"停止"按钮，终止进程。
3. **编辑配置**：点击右上角"编辑配置"或"新建项目"按钮进行管理。
4. **查看日志**：界面下方实时显示命令输出。

## 注意事项

- 仅支持 Windows 系统。
- 依赖 PowerShell 环境。
- 修改配置后，运行中的项目会被停止。
